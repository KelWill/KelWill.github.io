Boing!

Description

An educational companion app designed for 5th and 6th science class, Boing was constructed as part of the Famo.us pre-beta program, and was debuted at their inaugral code release.

Tech Stack

We used Famo.us and Backbone.js, and linked up the app with a Firebase database for real-time data capabilities.

Homework Buddy

Description

Homework Buddy allows teachers to easily create, assign, and grade homework by simply pasting in an article and adding questions to it.

Tech Stack

I used Node.js and Express coupled with a MySql database and Passport.js for authentication on the backend. The front end was constructed using Backbone.js and Bootstrap.
